Well, we'll start by listin all availabe options in VirtualMe. First, all functions available on the terminal object
1:
terminal.log(text,tag,color,font,size)
Description: Logs the entered text into the terminal.Here's a good example.
terminal.log('A serious error ocurred.','strong','red','monospace','15px')
2:
terminal.space()
Description: Jumps the next content to be inserted in the next line. If it is not used, multiple content will appear on the same line.
3:
terminal.clear()
Description: Cleans the terminal.
4:
terminal.pushCursor()
Description: Pushes the blinking cursor to the begining of the next line.If it is not called when new lines are logged, it remains blinking where it was.
5:
terminal.cursorText(text)
Description: Inserts text to the blinking cursor. For example the default text is #_.
6:
terminal.text()
Description: Calling this function returns the text entered by the user in the textfield.
7:
terminal.type()
Description: This function inserts the textfield on the current line where commands are supposed to be typed. The earlier one where a command was entered vanishes and a new one is displayed on the current line. It is called when you want the user to enter a command. If it is not updated when content is logged, it remais where it was as new data is entered.
8:
terminal.loadScript(path,onerror,onload)
Description: This loads the script in the specified path. 'onload' is the function to be called after the script is successfully loaded.'onerror' is the function to be called if the script is unsuccessfully loaded.
9:
terminal.onKey(func)
Description: The specified function will be called every time a key is pressed on the keyboard.
10:
terminal.getText()
Description: returns the text entered by the user in the textfield.
